# Spec: Dashboard & Analytics

## MODIFIED Requirements [Student Management System]

### Requirement: Trực quan hóa dữ liệu nâng cao
#### Scenario: Xem biểu đồ phân bổ sinh viên
- **Given** Dữ liệu sinh viên và khoa đã được nạp đầy đủ.
- **When** Người dùng chọn Tab "Thống kê".
- **Then** Hệ thống hiển thị biểu đồ tròn biểu thị tỷ lệ sinh viên theo Khoa.
- **And** Hiển thị biểu đồ cột so sánh số lượng sinh viên Xuất sắc, Giỏi, Khá, Trung bình giữa các Khoa.
